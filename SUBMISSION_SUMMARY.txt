PROBLEM STATEMENT
================================================================================

Objective: Analyze and forecast coffee commodity price trends from Vietnamese 
news articles using automated extraction and statistical analysis techniques.

Research Questions:
1. What are the statistical characteristics of coffee prices over time?
2. How do prices vary at different time scales (daily, weekly, monthly)?
3. What can market sentiment reveal about price movements?
4. Which news sources provide reliable price information?

Data Source: Vietnamese news websites (85 outlets)
Analysis Period: 66 days (January 12 - March 18, 2026)

================================================================================
METHODOLOGY OVERVIEW
================================================================================

Phase 1: Web Scraping (Data Collection)
- Framework: Scrapy (Python)
- Targets: 85 Vietnamese news sources
- Output: 2,875 news articles

Phase 2: Data Processing (ETL)
- Tool: Python Pandas with regex pattern matching
- Algorithm: Extract prices from article headlines
- Patterns: "XXX.XXX đ/kg" and "XXX đ/kg"
- Validation: Filter to 50,000-200,000 đ/kg range
- Output: 496 valid price records (17.3% success rate)

Phase 3: Feature Engineering
- Date decomposition: Year, month, day, week, day_of_week
- Price dynamics: Daily change, percent change
- Technical indicators: MA7 (7-day moving average), MA14 (14-day)
- Categorization: 8 news categories identified
- Total features: 12 new columns created

Phase 4: Statistical Analysis
- Descriptive statistics: Mean, median, std dev, quartiles
- Time series aggregation: Daily, weekly, monthly levels
- Market analysis: Up/down/flat day distribution
- Source analysis: Top news outlets identified

Phase 5: Visualization & Reporting
- Tools: Matplotlib (300 DPI PNG), Pandas
- Outputs: 7 professional charts, comprehensive reports
- Documentation: 5 detailed markdown files

METHODOLOGY DETAILS
================================================================================

Price Extraction Algorithm:

1. Load raw CSV with article headlines
2. Search for price patterns:
   - Pattern 1: "XXX.XXX đ/kg" (e.g., "100.000 đ/kg")
   - Pattern 2: "XXX đ/kg" (e.g., "100000 đ/kg")
3. Extract numeric values using regex
4. Validate price range: 50,000 - 200,000 đ/kg
   (Domain knowledge: Coffee prices typically 80k-150k)
5. If valid, store and associate with article date
6. Result: 496 valid extractions from 2,875 articles

Feature Engineering:

1. Temporal Features
   - Extract year, month, day from date
   - Calculate week_of_month (weeks 1-4)
   - Calculate day_of_week (Mon-Sun)

2. Price Movement Features
   - price_change: Diff between consecutive days
   - price_pct_change: Percent change from previous day

3. Technical Indicators
   - MA7: 7-day moving average (short-term trend)
   - MA14: 14-day moving average (medium-term trend)

4. Categorization
   - Parse headline for keywords
   - Classify into 8 categories
   - Track news source

Statistical Analysis:

1. Descriptive Statistics
   - Mean, median, mode, min, max
   - Std dev, variance, coefficient of variation
   - Quartiles (Q1, Q2, Q3, Q4)

2. Time Series Aggregation
   - Group by date: Daily average prices
   - Group by week: Weekly aggregation
   - Group by month: Monthly comparison

3. Market Dynamics
   - Count up days (price increased)
   - Count down days (price decreased)
   - Count flat days (no change)
   - Calculate average change per type

4. Source Analysis
   - Count articles per source
   - Identify top 15 sources
   - Calculate coverage distribution
